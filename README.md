# EduHestim

Plateforme de gestion des emplois du temps et reservation des salles - HESTIM 2025-2026

## Stack
- Frontend : HTML / CSS / JavaScript
- Backend  : Node.js / Express.js (port 5000)
- Base     : MongoDB (port 27017)
- Analyse  : Python / Jupyter / Chart.js

## Lancer le projet
cd Backend && node server.js

## 7 KPIs integres
1. Taux occupation salles
2. Volume horaire enseignants
3. Taux de conflits
4. Creneau le plus demande
5. Taux modification EDT
6. Taux disponibilite salles
7. Nb moyen seances par classe
