🎬 Scénario pour la Vidéo

📝 Étape 1 : Connexion Admin

  1. Ouvrir Connexion.html
  2. Se connecter avec admin@hestim.ma / admin123
  3. Explorer le dashboard

➕ Étape 2 : Créer un Compte

  1. Aller sur "Créer Utilisateur"
  2. Créer un nouvel étudiant
  3. Se déconnecter puis reconnecter avec le nouveau compte

📋 Étape 3 : Demande de Réservation

  1. Connecté en tant qu'étudiant
  2. Aller sur "Faire une Demande" → remplir et soumettre
  3. Aller sur "Mes Demandes" → voir "En attente"

✅ Étape 4 : Admin Traite la Demande

  1. Reconnecter en admin → "Gestion des Demandes"
  2. Approuver avec commentaire
  3. Reconnecter en étudiant → voir statut "Approuvée"
